# 👥 คู่มือโหมดติดตามคู่ (Dual Tracking Mode)

## 🎯 ฟีเจอร์ใหม่

ตอนนี้คุณสามารถ**ติดตาม 2 สตรีมเมอร์พร้อมกัน**แบบแบ่งจอได้แล้ว!

---

## 🚀 วิธีใช้งาน

### 1. เลือกโหมด
เมื่อเข้าหน้า Live จะเห็นปุ่มเลือกโหมด 2 แบบ:
- **👤 โหมดเดี่ยว** - ติดตาม 1 คน (เหมือนเดิม)
- **👥 โหมดคู่** - ติดตาม 2 คนพร้อมกัน (ใหม่!)

### 2. กรอก Username
**โหมดคู่:**
```
คนที่ 1: @username1    VS    คนที่ 2: @username2
```

### 3. เริ่มติดตาม
กดปุ่ม **"🚀 เริ่มติดตาม 2 คนพร้อมกัน"**

## ⚙️ การทำงาน

### ข้อมูลแยกอิสระ
- แต่ละคนมีข้อมูลแยกกัน (data1, data2)
- ไม่ปนกัน ดูได้ชัดเจน

### อัพเดทแบบ Real-Time
- ทั้ง 2 คนอัพเดทพร้อมกันทุก 1 วินาที
- แสดงข้อมูลได้สูงสุด **200 รายการ/คน**

### หยุดติดตาม
- กดปุ่ม "หยุดติดตาม" จะหยุดทั้ง 2 คนพร้อมกัน

---

## 💡 กรณีใช้งาน

### 1. เปรียบเทียบสตรีมเมอร์
ดูว่าใครได้รับของขวัญ/คอมเมนต์มากกว่ากัน

### 2. ติดตามคู่แข่ง
ดูสตรีมเมอร์ 2 คนที่แข่งกันอยู่

### 3. ติดตามเพื่อน
ดูเพื่อน 2 คนที่ไลฟ์พร้อมกัน

### 4. วิเคราะห์ข้อมูล
เปรียบเทียบ engagement ของ 2 สตรีมเมอร์

### Data Limit
- เก็บข้อมูลสูงสุด **200 รายการ/ประเภท/คน**
- รวมทั้งหมด: 200 × 4 ประเภท × 2 คน = **1,600 รายการ**

## ⚠️ ข้อควรระวัง

### 1. ประสิทธิภาพ
- ติดตาม 2 คนพร้อมกัน = ใช้ทรัพยากรเพิ่มขึ้น 2 เท่า
- แนะนำให้ใช้บนคอมพิวเตอร์ที่มีสเปคดี

### 2. Network
- ใช้ bandwidth มากขึ้น (2 API calls/วินาที)
- ต้องมีอินเทอร์เน็ตเร็วและเสถียร

### 3. TikTok API
- อาจโดน rate limit ถ้าเรียก API บ่อยเกินไป
- ถ้าเกิดปัญหา ให้ลดความถี่ในการ fetch
